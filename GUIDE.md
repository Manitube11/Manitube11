# راهنمای استفاده از ربات معامله‌گر مانی‌تیوب (نسخه گرافیکی)

این نسخه جدید و پیشرفته ربات است که دارای **رابط کاربری گرافیکی (GUI)** بوده و از **داده‌های واقعی بازار** استفاده می‌کند.

## نصب پیش‌نیازها

برای اجرای این برنامه، نیاز به اینترنت دارید تا قیمت‌ها را دریافت کنید.
ابتدا دستور زیر را در ترمینال اجرا کنید تا کتابخانه دریافت قیمت نصب شود:

```bash
pip install yfinance pandas requests
```

(معمولاً `tkinter` که برای گرافیک است به صورت پیش‌فرض روی پایتون نصب است).

---

## اجرای برنامه

1. روی فایل `trading_bot/main.py` کلیک راست کرده و آن را با پایتون اجرا کنید.
2. یا در ترمینال دستور زیر را بزنید:
   ```bash
   python trading_bot/main.py
   ```

---

## راهنمای پنل کاربری

### تب تنظیمات (Settings)
در اولین اجرا، به تب **Settings** بروید:
1. **Bot Token:** توکن ربات تلگرام خود را وارد کنید.
2. **Chat ID:** آی‌دی عددی خود یا کانال را وارد کنید.
   - اگر آی‌دی را نمی‌دانید، فایل `get_chat_id.py` را اجرا کنید.
3. دکمه **Save & Connect** را بزنید.
4. دکمه **Test Message** را بزنید. اگر پیامی در تلگرام دریافت کردید، همه چیز آماده است!

### تب داشبورد (Dashboard)
1. دکمه **START TRADING** را بزنید.
2. ربات شروع به دریافت قیمت‌های زنده (بیت‌کوین، اتریوم، طلا، نفت) می‌کند.
3. جدول به صورت خودکار آپدیت می‌شود و تحلیل‌ها (RSI, SMA) را نشان می‌دهد.
4. اگر سیگنال خرید یا فروشی شناسایی شود:
   - در قسمت Logs نمایش داده می‌شود.
   - فوراً به تلگرام شما ارسال می‌شود.

---

## نکات مهم
- این برنامه از اینترنت استفاده می‌کند، پس اتصال خود را چک کنید.
- داده‌ها هر ۱ دقیقه یکبار آپدیت می‌شوند تا اینترنت شما زیاد مصرف نشود.
- اگر برنامه باز نشد، مطمئن شوید پایتون شما نسخه جدید است.
