# راهنمای استفاده از سیستم معاملاتی هوشمند VIP مانی‌تیوب

تبریک! شما اکنون به نسخه **حرفه‌ای و VIP** سیستم معاملاتی دسترسی دارید. این نسخه بازنویسی شده است تا مانند صندوق‌های سرمایه‌گذاری (Hedge Funds) عمل کند.

---

## ویژگی‌های جدید نسخه VIP
1. **پوشش ۱۲ دارایی مهم:** شامل بیت‌کوین، اتریوم، آلت‌کوین‌های برتر (Solana, BNB, ADA...)، طلا و نفت.
2. **هوش مصنوعی پیشرفته:** تحلیل همزمان روند (Trend)، مومنتوم (MACD)، قدرت (RSI) و نوسان (ATR).
3. **مدیریت ریسک خودکار:** محاسبه دقیق حد ضرر (Stop Loss) و حد سود (Take Profit) بر اساس نوسانات لحظه‌ای بازار.
4. **سیستم فال‌بک (Fallback):** اگر اینترنت قطع شود، سیستم به صورت خودکار وارد حالت شبیه‌سازی می‌شود تا عملکرد خود را تست کند.

---

## نصب و راه‌اندازی

دستور زیر را در ترمینال اجرا کنید تا کتابخانه‌های لازم نصب شوند:
```bash
pip install yfinance pandas requests numpy
```
*(حتماً `sudo apt-get install python3-tk` را در لینوکس اگر نیاز بود اجرا کنید)*

## راهنمای پنل کاربری

### ۱. تب Config (تنظیمات)
قبل از هر کاری، ربات تلگرام خود را متصل کنید تا سیگنال‌های VIP را روی گوشی دریافت کنید:
1. **Bot Token:** توکن ربات خود را وارد کنید.
2. **Chat ID:** آی‌دی عددی خود را وارد کنید.
3. دکمه **Save & Connect** را بزنید و سپس با **Test Message** اتصال را تست کنید.

### ۲. تب Live Market (بازار زنده)
این مرکز فرماندهی شماست:
- دکمه **ACTIVATE AI** را بزنید تا موتور هوشمند روشن شود.
- **جدول تحلیل:**
  - **Trend:** جهت کلی بازار (UP ↗ یا DOWN ↘) بر اساس میانگین‌های متحرک نمایی (EMA 50/200).
  - **MACD & RSI:** شاخص‌های قدرت روند.
  - **ATR:** شاخص نوسان برای محاسبه حد ضرر.
  - **Signal:** وضعیت فعلی (WAIT, BUY, SELL).

### ۳. سیگنال‌های VIP
وقتی یک موقعیت معاملاتی عالی پیدا شود، ربات پیامی با فرمت زیر به تلگرام می‌فرستد:

```text
💎 VIP SIGNAL: BTC-USD
--------------------------------
Type: 🟢 BUY (LONG)
Entry: $95,430.00

🎯 Targets:
TP1: $97,000 (Conservative)
TP2: $99,500 (Growth)
TP3: $105,000 (Moon bag)

🛑 Stop Loss: $93,100
⚖️ Risk: MEDIUM
--------------------------------
Reason: Trend UP + MACD Cross + Vol Confirmation
```

## نکته تکنیکال
این سیستم از **yfinance** برای دریافت قیمت‌های زنده استفاده می‌کند. اگر به هر دلیلی (مثل تحریم یا قطعی اینترنت) نتواند به سرورهای یاهو وصل شود، به طور خودکار داده‌های شبیه‌سازی شده تولید می‌کند تا شما بتوانید منطق برنامه را تست کنید.
