# راهنمای استفاده از موتور معاملاتی مانی‌تیوب (ManiTube Trading Engine)

این یک نمونه اولیه حرفه‌ای از یک سیستم معاملاتی الگوریتمی است که با استفاده از پایتون ساخته شده است. این سیستم قیمت‌های بیت‌کوین را شبیه‌سازی کرده و با استفاده از استراتژی "تقاطع طلایی" (Golden Cross) و شاخص RSI معامله می‌کند.

## ویژگی‌ها

- **شبیه‌سازی بازار:** تولید داده‌های واقعی قیمت بیت‌کوین.
- **تحلیل تکنیکال:** محاسبه میانگین متحرک (SMA) و شاخص قدرت نسبی (RSI).
- **مدیریت ریسک:** دارای حد ضرر (Stop Loss) و حد سود (Take Profit) خودکار.
- **اتصال به تلگرام:** قابلیت ارسال سیگنال‌های خرید و فروش به کانال تلگرام شما.
- **گزارش‌دهی:** نمایش دفتر کل معاملات و خلاصه عملکرد (سود/زیان).

1. **نصب کتابخانه‌ها:**
   ترمینال یا Command Prompt را باز کنید و دستور زیر را اجرا کنید:


## نحوه اجرا

1. فایل‌های پروژه را در یک پوشه قرار دهید.
2. ترمینال را در آن پوشه باز کنید.
3. دستور زیر را برای اجرای برنامه وارد کنید:
   ```bash
   python trading_bot/main.py
   ```
   *(توجه: اگر در ویندوز هستید ممکن است نیاز باشد از `python` به جای `python3` استفاده کنید)*

---

## راهنمای اتصال به تلگرام

برای اینکه برنامه بتواند سیگنال‌ها را به کانال تلگرام شما بفرستد، مراحل زیر را انجام دهید:

1. **ساخت ربات:**
   - در تلگرام به آیدی `@BotFather` پیام دهید.
   - دستور `/newbot` را ارسال کنید.
   - نام و نام کاربری ربات را انتخاب کنید.
   - در نهایت یک **Token** به شما می‌دهد (مثلاً `123456789:ABCDefGhIjK...`).

2. **دریافت Chat ID:**
   - ربات خود را به عنوان مدیر (Admin) در کانال خود اضافه کنید.
   - یک پیام تستی در کانال بفرستید.
   - از این آدرس در مرورگر استفاده کنید (به جای `YOUR_TOKEN` توکن خود را بگذارید):
     `https://api.telegram.org/botYOUR_TOKEN/getUpdates`
   - در پاسخ JSON، مقدار `chat` -> `id` را پیدا کنید (معمولاً با -100 شروع می‌شود).

3. **تنظیم برنامه:**
   - فایل `trading_bot/main.py` را با یک ویرایشگر متن (مثل Notepad) باز کنید.
   - مقادیر زیر را پیدا کرده و جایگزین کنید:
     ```python
     TELEGRAM_BOT_TOKEN = "توکن_خود_را_اینجا_بگذارید"
     TELEGRAM_CHAT_ID = "آیدی_کانال_را_اینجا_بگذارید"
     ```

4. **ذخیره و اجرا:**
   - فایل را ذخیره کنید و دوباره برنامه را اجرا کنید. اکنون سیگنال‌ها به تلگرام شما ارسال می‌شوند.
