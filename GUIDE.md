# راهنمای سیستم معاملاتی هوشمند VIP مانی‌تیوب (نسخه PRO 3.0)

این نسخه جدید با ظاهر گرافیکی مدرن (Dark Mode) و قابلیت‌های پیشرفته هوش مصنوعی آماده شده است.

---

## 🚀 ویژگی‌های جدید
1. **رابط کاربری حرفه‌ای:** طراحی جدید با تم تاریک و دکمه‌های شیشه‌ای.
2. **ارسال تصویر نمودار:** همراه هر سیگنال، تصویر تحلیل تکنیکال نمودار هم ارسال می‌شود.
3. **فرمت Cornix:** سیگنال‌ها دقیقا با فرمت استاندارد ربات‌های کپی‌تریدینگ ارسال می‌شوند.
4. **هشدار نهنگ:** تشخیص حجم‌های غیرعادی (Whale Alert).
5. **پشتیبانی از ۳۲+ دارایی:** شامل بیت‌کوین، تمام آلت‌کوین‌های مهم، طلا، نفت و سهام آمریکا.

---

## 📥 نصب و راه‌اندازی (مهم)
چون قابلیت‌های گرافیکی اضافه شده است، باید کتابخانه‌های جدید را نصب کنید:

1. دکمه `Windows + R` را بزنید، بنویسید `cmd` و اینتر کنید.
2. دستور زیر را کپی کرده و در صفحه سیاه پیست کنید (و اینتر بزنید):

```bash
pip install customtkinter mplfinance yfinance pillow requests pandas numpy
```

*(منتظر بمانید تا نصب تمام شود)*

---

## ▶ نحوه اجرا
بعد از نصب، برای اجرای برنامه دستور زیر را وارد کنید:

```bash
python trading_bot/gui_app.py
```

---

## 🛠 راهنمای حل مشکل تلگرام
اگر پیام‌ها به کانال نمی‌رسد:
1. در تب **Configuration**، دکمه **Force Test Signal** را بزنید.
2. اگر ارور داد، توکن ربات یا آیدی کانال اشتباه است.
3. ربات باید در کانال شما **Admin** باشد.
