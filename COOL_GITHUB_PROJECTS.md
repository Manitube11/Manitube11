# پروژه‌های جالب و محبوب در گیت‌هاب (Cool GitHub Projects)

در اینجا لیستی از چند پروژه معروف و کاربردی در گیت‌هاب آورده شده است که در جامعه برنامه‌نویسی بسیار محبوب هستند:

## 1. AutoGPT
**توضیحات:** یک پروژه تجربی و متن‌باز که هدف آن ساخت ایجنت‌های کاملاً خودکار با استفاده از GPT-4 است. این ابزار می‌تواند به صورت خودکار تسک‌ها را مدیریت و اجرا کند.
- [لینک ریپازیتوری](https://github.com/Significant-Gravitas/Auto-GPT)

## 2. LangChain
**توضیحات:** فریم‌ورکی قدرتمند برای توسعه اپلیکیشن‌هایی که از مدل‌های زبانی بزرگ (LLM) استفاده می‌کنند. این ابزار به شما کمک می‌کند تا زنجیره‌ای از درخواست‌ها را برای مدل‌های هوش مصنوعی ایجاد کنید.
- [لینک ریپازیتوری](https://github.com/hwchase17/langchain)

## 3. Supabase
**توضیحات:** یک جایگزین متن‌باز برای Firebase که تمام ابزارهای لازم برای توسعه بک‌اند (مانند دیتابیس، احراز هویت، استوریج و غیره) را فراهم می‌کند.
- [لینک ریپازیتوری](https://github.com/supabase/supabase)

## 4. Tauri
**توضیحات:** ابزاری برای ساخت اپلیکیشن‌های دسکتاپ کوچک‌تر، سریع‌تر و امن‌تر با استفاده از تکنولوژی‌های فرانت‌اند وب. جایگزینی سبک‌تر برای Electron.
- [لینک ریپازیتوری](https://github.com/tauri-apps/tauri)

## 5. Excalidraw
**توضیحات:** یک تخته سفید مجازی (Virtual Whiteboard) برای کشیدن دیاگرام‌ها و طرح‌هایی که شبیه به نقاشی دستی به نظر می‌رسند. بسیار محبوب برای جلسات آنلاین و مستندسازی.
- [لینک ریپازیتوری](https://github.com/excalidraw/excalidraw)

## 6. Bun
**توضیحات:** یک ران‌تایم جاوا اسکریپت جدید و فوق‌العاده سریع که شامل باندلر، ترنسپایلر و پکیج منیجر هم می‌شود و رقیبی جدی برای Node.js و Deno است.
- [لینک ریپازیتوری](https://github.com/oven-sh/bun)

## 7. Vite
**توضیحات:** ابزار بیلد نسل جدید برای فرانت‌اند که سرعت توسعه را بسیار بالا می‌برد و تجربه برنامه‌نویسی مدرن و روانی را فراهم می‌کند.
- [لینک ریپازیتوری](https://github.com/vitejs/vite)

## 8. The Art of Command Line
**توضیحات:** یک راهنمای جامع و بسیار کاربردی برای تسلط بر خط فرمان (Command Line) که نکات و ترفندهای زیادی را در یک صفحه گردآوری کرده است.
- [لینک ریپازیتوری](https://github.com/jlevy/the-art-of-command-line)
