# ManiTube AI Trading Bot - PRO 3.0

This is a Python-based trading signal generator featuring a modern GUI, real-time market data analysis, and professional Telegram alerts.

## Features
- **Modern GUI:** Built with `customtkinter` for a sleek dark-themed experience.
- **Multi-Asset Support:** Crypto, Forex, Commodities, and Stocks.
- **Visual Signals:** Generates and sends candlestick charts with indicators via Telegram.
- **AI Analysis:** Trend Following, MACD, RSI, ATR Risk Management, and Volume Spike detection.
- **Cornix Ready:** Signals are formatted for auto-trading bot compatibility.

## Quick Start
1. Install dependencies:
   ```bash
   pip install customtkinter mplfinance yfinance pillow requests pandas numpy
   ```
2. Run the application:
   ```bash
   python trading_bot/gui_app.py
   ```

## Configuration
- Launch the app and go to the **Configuration** tab.
- Enter your Telegram Bot Token and Chat ID.
- Add or Remove assets from the watchlist dynamically.
