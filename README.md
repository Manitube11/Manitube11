# اسکنر هوشمند ایپی تمیز و تانل‌ یاب خودکار (V2.3)
# Cloudflare & GCore Tunnel & SNI Discovery

این پیشرفته‌ترین نسخه برنامه است که مخصوص شرایط فیلترینگ شدید SNI در ایران طراحی شده است.

## قابلیت‌های جدید (نسخه ۲.۳)
- **تانل‌ یاب خودکار (IR Bridge Discovery):** جستجو در میان هزاران کانفیگ برای پیدا کردن سرورهایی که تانل ایران هستند (پایدارترین روش فعلی).
- **اسکنر SNI (Domain Discovery):** تست خودکار دامنه‌های مختلف روی اینترنت شما برای پیدا کردن دامنه‌هایی که توسط اپراتور مسدود نشده‌اند.
- **تکنیک جایگزینی SNI:** اگر دامنه‌ی اصلی یک کانفیگ مسدود باشد، برنامه به صورت خودکار از یک دامنه‌ی امن (مثل Skype یا Microsoft) استفاده می‌کند تا اتصال برقرار شود.
- **فرگمنت تصادفی (Random Fragment):** استفاده از پارامترهای تصادفی در فرگمنت برای جلوگیری از شناسایی توسط DPI.
- **پشتیبانی از Gcore:** اسکن همزمان شبکه‌های کلودفلر و جیکور.

## نحوه استفاده
فایل `main.py` را اجرا کنید:
```bash
python main.py
```
گزینه **۲ (Automatic)** را انتخاب کنید. برنامه در ابتدا دامنه‌های باز را روی اینترنت شما پیدا می‌کند و سپس کانفیگ‌های مناسب را بر اساس آن‌ها می‌سازد و تست می‌کند.
