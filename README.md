# اسکنر ایپی تمیز کلودفلر و سازنده کانفیگ وی‌توری
# Cloudflare Clean IP Scanner & V2Ray Configurator

این برنامه برای پیدا کردن ایپی‌های تمیز کلودفلر که در ایران مسدود نیستند و ساخت خودکار کانفیگ‌های V2Ray (VLESS/VMess) طراحی شده است.

## امکانات
- دریافت خودکار آخرین رنج‌های ایپی کلودفلر.
- اسکن چند‌رشته‌ای (Multi-threaded) برای پیدا کردن کمترین تاخیر (Latency).
- قابلیت چک کردن هاست (Host Check) برای اطمینان از اتصال به ورکر یا دامنه خاص.
- ساخت خودکار لینک‌های VLESS و VMess مخصوص کلودفلر.
- رابط کاربری ساده به زبان فارسی.

## پیش‌نیازها
برای اجرای این برنامه به پایتون ۳ و کتابخانه `requests` نیاز دارید:

```bash
pip install requests
```

## نحوه اجرا
کافیست فایل `main.py` را اجرا کنید:

```bash
python main.py
```

پس از اجرا، مراحل زیر را دنبال کنید:
1. تعداد نمونه در هر رنج را مشخص کنید (پیش‌فرض ۵).
2. سرعت اسکن را انتخاب کنید (تعداد تردها).
3. اطلاعات کانفیگ خود را وارد کنید (UUID, Host, Path).
4. در نهایت، ۳ کانفیگ برتر با کمترین تاخیر برای شما نمایش داده می‌شود.

### بخش فنی
- `scanner.py`: وظیفه دریافت رنج‌ها و تست ایپی‌ها را دارد.
- `config_generator.py`: وظیفه تولید لینک‌های استاندارد V2Ray را دارد.
- `main.py`: رابط کاربری و مدیریت جریان برنامه.
