# اسکنر ایپی تمیز کلودفلر و سازنده کانفیگ وی‌توری
# Cloudflare Clean IP Scanner & V2Ray Configurator

این برنامه برای پیدا کردن ایپی‌های تمیز کلودفلر که در ایران مسدود نیستند و ساخت یا اصلاح خودکار کانفیگ‌های V2Ray (VLESS/VMess) طراحی شده است.

## امکانات جدید
- **حالت خودکار (I have nothing):** اگر سرور یا کانفیگ ندارید، برنامه خودش کانفیگ‌های عمومی را پیدا کرده و با ایپی تمیز برای شما اصلاح می‌کند.
- **اصلاح هوشمند (Patching):** جایگذاری ایپی تمیز در کانفیگ‌های VLESS و VMess با حفظ تنظیمات SNI و Host.
- دریافت خودکار آخرین رنج‌های ایپی کلودفلر.
- اسکن چند‌رشته‌ای (Multi-threaded) برای پیدا کردن کمترین تاخیر (Latency).
- رابط کاربری ساده به زبان فارسی.

## پیش‌نیازها
برای اجرای این برنامه به پایتون ۳ و کتابخانه `requests` نیاز دارید:

```bash
pip install requests
```

## نحوه اجرا
کافیست فایل `main.py` را اجرا کنید:

```bash
python main.py
```

پس از اجرا، دو گزینه دارید:
1. **ساخت کانفیگ با مشخصات سرور خودم:** اگر خودتان سرور یا پنل دارید، مشخصات را وارد می‌کنید تا با ایپی تمیز لینک بسازد.
2. **حالت خودکار:** برنامه خودش همه کارها را انجام می‌دهد و در نهایت چند لینک آماده اتصال به شما می‌دهد.

---

### بخش فنی
- `scanner.py`: وظیفه دریافت رنج‌ها و تست ایپی‌ها را دارد.
- `config_generator.py`: وظیفه تولید لینک‌های خام را دارد.
- `config_patcher.py`: وظیفه دریافت کانفیگ‌های عمومی و اصلاح (Patch) آن‌ها با ایپی تمیز را دارد.
- `main.py`: رابط کاربری و مدیریت جریان برنامه.
