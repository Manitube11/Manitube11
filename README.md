# اسکنر هوشمند ایپی تمیز و تست‌گر زنده اتصال (V2.2)
# Cloudflare & GCore Smart Scanner + Live Verification

این ابزار پیشرفته در نسخه ۲.۲ مجهز به سیستم تست زنده (Live Test) شده است که قبل از نمایش هر کانفیگ، اتصال آن را مستقیماً با اینترنت شما بررسی می‌کند.

## قابلیت‌های جدید (نسخه ۲.۲)
- **تست زنده اتصال (Live Verification):** هر کانفیگ قبل از نمایش، یک بار با اینترنت شما تست TLS می‌شود. فقط مواردی که علامت `[✔] VERIFIED` دارند نمایش داده می‌شوند.
- **اسکن دو مرحله‌ای:** مرحله اول پیدا کردن ایپی‌های پینگ‌دار و مرحله دوم تست پایداری اتصال با SNI مخصوص هر کانفیگ.
- **پشتیبانی کامل از Fragment:** تمامی کانفیگ‌های تولیدی دارای تنظیمات بهینه فرگمنت برای دور زدن فیلترینگ هستند.
- **حالت خودکار هوشمند:** پیدا کردن بهترین کانفیگ‌های عمومی و شخصی‌سازی آن‌ها با ایپی‌های تمیزی که با نت شما سازگارند.

## پیش‌نیازها
```bash
pip install -r requirements.txt
```

## نحوه استفاده
فایل `main.py` را اجرا کنید:
```bash
python main.py
```
برنامه در مرحله دوم (Phase 2) شروع به تست کانفیگ‌ها می‌کند. اگر کانفیگی نمایش داده شد، یعنی با اینترنت فعلی شما سازگار است.
