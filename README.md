# اسکنر هوشمند ایپی تمیز و الگوی کانفیگ پولی (V2.6.1)
# Professional Iran-Pattern Scanner & Tunnels (v2.6.1)

این نسخه به صورت اختصاصی بر اساس الگوهای کانفیگ‌های پولی و تانل‌های حرفه‌ای ایران بازنویسی شده است.

## قابلیت‌های جدید (نسخه ۲.۶.۱)
- **رفع مشکل اتصال (Fix Routing):** جداسازی SNI و Host Header برای کارکرد صحیح در کلودفلر (فیلترینگ دور زده می‌شود اما مسیریابی سالم می‌ماند).
- **الگوی کانفیگ پولی:** استفاده از دامنه‌های سفید شده ایرانی (`skyroom.online`, `gharar.ir`, `igap.net`, `snapp.ir`) برای فریب سیستم فیلترینگ (SNI Spoofing).
- **پشتیبانی از HTTP Header Spoofing:** ساخت خودکار کانفیگ‌های TCP با هدر HTTP جعلی (مشابه تانل‌های پولی) برای عبور از فیلترینگ اپراتورها.
- **شناسایی تانل مستقیم ایران:** پیدا کردن خودکار ایپی‌های دیتاسنترهای داخلی (آسیاتک، زیرساخت و ...) که به عنوان پل (Bridge) عمل می‌کنند.
- **ترکیب با Reality:** نمایش همزمان قوی‌ترین کانفیگ‌های ریالیتی موجود.

## نحوه استفاده (ویندوز)
کافیست فایل `run.bat` را اجرا کنید. برنامه به صورت خودکار پیش‌نیازها را نصب کرده و اجرا می‌شود.

## نحوه استفاده (لینوکس/اندروید)
```bash
pip install -r requirements.txt
python main.py
```
سپس گزینه **۲ (Auto)** را انتخاب کنید. برنامه ابتدا دامنه‌های باز را تست کرده و سپس کانفیگ‌هایی با الگوی نمونه‌های پولی برای شما می‌سازد.
