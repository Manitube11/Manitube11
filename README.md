## قابلیت‌های جدید (نسخه ۲.۵)
- **الگوی کانفیگ پولی:** استفاده از دامنه‌های سفید شده ایرانی (`skyroom.online`, `gharar.ir`, `igap.net`, `snapp.ir`) برای فریب سیستم فیلترینگ (SNI Spoofing).

- **شناسایی تانل مستقیم ایران:** پیدا کردن خودکار ایپی‌های دیتاسنترهای داخلی (آسیاتک، زیرساخت و ...) که به عنوان پل (Bridge) عمل می‌کنند.
- **ترکیب با Reality:** نمایش همزمان قوی‌ترین کانفیگ‌های ریالیتی موجود.
