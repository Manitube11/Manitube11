# اسکنر حرفه‌ای ایپی تمیز و اصلاح‌گر خودکار کانفیگ (V2.1)
# Professional Clean IP Scanner & Config Patcher

این ابزار پیشرفته برای دور زدن محدودیت‌های شدید اینترنت در ایران طراحی شده است. برخلاف ابزارهای معمولی، این برنامه از اسکن عمیق TLS برای تایید ۱۰۰٪ سلامت ایپی استفاده می‌کند.

## قابلیت‌های جدید (نسخه ۲.۱)
- **اسکن عمیق (Deep Scan):** انجام دست‌دهی واقعی TLS (Handshake) برای اطمینان از مسدود نبودن SNI روی ایپی.
- **پشتیبانی از Fragment:** اضافه کردن خودکار تنظیمات فرگمنت به کانفیگ‌ها برای عبور از سیستم DPI.
- **ایپی‌های Gcore:** اضافه شدن رنج‌های ایپی Gcore علاوه بر کلودفلر.
- **اصلاح هوشمند (Auto Patching):** پیدا کردن خودکار کانفیگ‌های عمومی و شخصی‌سازی آن‌ها با ایپی‌های تمیز.

## پیش‌نیازها
```bash
pip install -r requirements.txt
```

## نحوه استفاده
فایل `main.py` را اجرا کنید:
```bash
python main.py
```
سپس بین حالت **دستی (Manual)** و **خودکار (Automatic)** یکی را انتخاب کنید.

**نکته مهم:** برای استفاده از حداکثر کارایی، در برنامه کلاینت خود (مثل v2rayNG یا Nekobox) حتما قابلیت **Fragment** را فعال کنید.
